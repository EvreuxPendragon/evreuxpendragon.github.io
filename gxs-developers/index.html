Adobe Illustrator ExtendScript Challenge

Please read carefully the instructions below, tasks #1 and #3 have specific requirements inside, click them to expand further information.

Task 1: Create a new Illustrator document with the following properties:

- Width: 800 points
- Height: 600 points
- Color Mode: RGB
- Resolution: 300 DPI

Task 2: Create a text frame with the text "Illustrator Challenge" and place it at coordinates (100, 100) on the artboard.
Task 3: Create a rectangle with the following properties:

- Width: 200 points
- Height: 100 points
- Fill Color: Red
- Stroke Color: Black
- Place it at coordinates (300, 300) on the artboard.

Task 4: Create a new layer called "Shapes" and move the rectangle to this layer.
Task 5: Create a new folder in the Layers panel called "Text" and move the text frame to this folder.
Task 6: Create a new folder in the Layers panel called "Artwork" and move both the "Shapes" layer and the "Text" folder to this folder.
Task 7: Iterate through all the layers in the "Artwork" folder and change the fill color of any shapes to blue.
Task 8: Browse a specific folder on your computer and load an image file (e.g., PNG or JPEG) into the Illustrator document. Place it at coordinates (400, 400) on the artboard.
Task 9: Save the Illustrator document as both an AI (Adobe Illustrator) and SVG (Scalable Vector Graphics) file with the name "Illustrator_Challenge."
Task 10: Export the entire artboard as a high-resolution PNG image with a transparent background.
Task 11: Create a new text frame with the current date and time and place it at the bottom right corner of the artboard.
Task 12: Close the Illustrator document without saving any changes.

Please remember to export / save your set of actions so we can test it in any computer. Good luck and thank you for your time!
