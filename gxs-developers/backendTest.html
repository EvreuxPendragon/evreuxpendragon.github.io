Back-End Developer Test

This back-end development test is designed to assess your skills working with APIs and databases. The test will evaluate your ability to identify layout issues and solve them accurately.

Some suggestions before you begin:

- Read the instructions carefully.
- Using Visual Studio Code as editor is not mandatory but recommended.
- Using the Live Server extension may help you debug your code.
- Use Dev Tools on your browser to debug your code.
- Make sure that you are working in the backendTest.html file.
- Styling is not required for this test, but if you wish to style what you see, please do so.
- The last part of the test is combined with the Front-End test, so make sure to read both instructions before proceeding.

If you have any questions, feel free to reach out to me at mario.pon@graphxsource.com.

Section 02

This part will simulate you saving data on to a database. Since we don't currently have a database we will create json objects instead.

The following form will have 3 inputs: Name, Age, and Email.

When the submit button gets clicked, 3 things should happen:
1. We should validate the data to make sure we're not adding the same person more than once.
2. An employee ID shoudl be generated if the data is valid.
3. We should add the data to the database. Or in our case, the json object.
