Front-End Developer Test

This front-end development test is designed to assess your skills in markup writing, CSS styling, and responsiveness. The test will evaluate your ability to identify layout issues and solve them accurately.

Some suggestions before you begin:

- Read the instructions carefully.
- Using Visual Studio Code as editor is not mandatory but recommended.
- Using the Live Server extension may help you debug your code.
- Use Dev Tools on your browser to debug your code.
- Make sure that you are working in the index.html file.
- Styling is intended to be made using pure CSS but SCSS is allowed if you're more comfortable with it.
- The last part of the test is combined with the Back-End test, so make sure to read both instructions before proceeding.

Section 03

In this final section of the test, you must validate the information provided by the user, in this case, make sure that the selected name exists, the age is between 18 and 99, and the email address is valid. You should achieve this using vanilla Javascript but jQuery is also allowed.

After validating the data, you must connect to a database (Back-End Test) and retrieve the remaining information to finally display it on the card below the form.

Your scripts are already linked to this file, you will only need to modify the test.js file in this directory.
